# 客户端入口与配置

## 安装

Packagist 包：

- `taohuahub/ai2-sdk-php`
- https://packagist.org/packages/taohuahub/ai2-sdk-php

安装命令：

```bash
composer require taohuahub/ai2-sdk-php
```

安装指定稳定版本：

```bash
composer require taohuahub/ai2-sdk-php:^0.1
```

## 命名空间入口

- `\TaohuaHub\Ai2\merchant(array|MerchantConfig $config): MerchantClient`
- `\TaohuaHub\Ai2\system(array|SystemConfig $config): SystemClient`

## MerchantConfig

类：`TaohuaHub\Ai2\Config\MerchantConfig`

构造参数：

- `baseUri`
- `mchNo`
- `sourceSystem`
- `accessSecret`
- `timeout = 10.0`
- `requestIdPrefix = 'req'`

辅助方法：

- `MerchantConfig::fromArray(array $config): self`
- `isValid(): bool`
- `errorCode(): string`
- `errorMessage(): string`

数组键：

- `base_uri`
- `mch_no`
- `source_system`
- `access_secret`
- `timeout`
- `request_id_prefix`

## SystemConfig

类：`TaohuaHub\Ai2\Config\SystemConfig`

构造参数：

- `baseUri`
- `systemKey`
- `timeout = 10.0`
- `requestIdPrefix = 'req'`

辅助方法：

- `SystemConfig::fromArray(array $config): self`
- `isValid(): bool`
- `errorCode(): string`
- `errorMessage(): string`

数组键：

- `base_uri`
- `system_key`
- `timeout`
- `request_id_prefix`

## MerchantClient

类：`TaohuaHub\Ai2\Client\MerchantClient`

公开方法：

- `sessions(): TaohuaHub\Ai2\Resource\Merchant\Sessions`
- `session(string $sessionId): TaohuaHub\Ai2\Context\Merchant\SessionContext`
- `messages(): TaohuaHub\Ai2\Resource\Merchant\Messages`
- `jobs(): TaohuaHub\Ai2\Resource\Merchant\Jobs`
- `stats(): TaohuaHub\Ai2\Resource\Merchant\Stats`
- `health(): ApiResult`
- `resolveUrl(string $path): string`

## SystemClient

类：`TaohuaHub\Ai2\Client\SystemClient`

公开方法：

- `sessions(): TaohuaHub\Ai2\Resource\System\Sessions`
- `session(string $sessionId): TaohuaHub\Ai2\Context\System\SessionContext`
- `messages(): TaohuaHub\Ai2\Resource\System\Messages`
- `jobs(): TaohuaHub\Ai2\Resource\System\Jobs`
- `merchants(): TaohuaHub\Ai2\Resource\System\Merchants`
- `sourceSystems(): TaohuaHub\Ai2\Resource\System\SourceSystems`
- `planPolicies(): TaohuaHub\Ai2\Resource\System\PlanPolicies`
- `platforms(): TaohuaHub\Ai2\Resource\System\Platforms`
- `platformAccounts(): TaohuaHub\Ai2\Resource\System\PlatformAccounts`
- `models(): TaohuaHub\Ai2\Resource\System\Models`
- `agents(): TaohuaHub\Ai2\Resource\System\Agents`
- `ops(): TaohuaHub\Ai2\Resource\System\Ops`
- `health(): ApiResult`
- `resolveUrl(string $path): string`

## 公共辅助接口

- `health()` 对应 `GET /healthz`
- `resolveUrl()` 用于把相对 `sse_url` 拼成完整地址

## 示例

```php
<?php

require_once __DIR__ . '/vendor/autoload.php';

use TaohuaHub\Ai2\Config\MerchantConfig;
use TaohuaHub\Ai2\Config\SystemConfig;

$merchant = \TaohuaHub\Ai2\merchant(new MerchantConfig(
    'https://example.com',
    'MCH202604110001',
    'default',
    'merchant-secret'
));

$system = \TaohuaHub\Ai2\system(new SystemConfig(
    'https://example.com',
    'system-api-key'
));
```
