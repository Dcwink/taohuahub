# Merchant 会话接口

## 集合资源

类：`TaohuaHub\Ai2\Resource\Merchant\Sessions`

### `create(array $payload): ApiResult`

- Endpoint: `POST /admin/merchant/v1/sessions/create`
- 主要参数：
  - `chat_mode`
  - `actor_type`
  - `actor_id`
  - `actor_name`
  - `conversation_key`
  - `business_type`
  - `business_id`
  - `isnew`
  - `agent_id`
  - `model_id`

### `getList(array $query = []): ApiResult`

- Endpoint: `GET /admin/merchant/v1/sessions`
- 常用查询参数：
  - `page`
  - `pagesize`
  - `s_session_id`
  - `s_actor_type`
  - `s_actor_id`
  - `s_conversation_key`
  - `s_chat_mode`
  - `s_status`

## 单会话上下文

类：`TaohuaHub\Ai2\Context\Merchant\SessionContext`

### `id(): string`

- 返回当前绑定的 `session_id`

### `get(): ApiResult`

- Endpoint: `GET /admin/merchant/v1/sessions/detail`

### `close(array $payload = []): ApiResult`

- Endpoint: `POST /admin/merchant/v1/sessions/close`
- SDK 会强制注入当前上下文的 `session_id`
- 可附加参数：`reason`

### `sseSign(array $payload = []): ApiResult`

- Endpoint: `POST /admin/merchant/v1/sessions/sse-sign`
- SDK 会强制注入当前上下文的 `session_id`
- 可附加参数：`expire_seconds`

### `switchModel(array $payload): ApiResult`

- Endpoint: `POST /admin/merchant/v1/sessions/switch-model`
- SDK 会强制注入当前上下文的 `session_id`
- 主要参数：`model_id`

### `messages(): SessionMessages`

- 进入当前会话的消息上下文

### `jobs(): SessionJobs`

- 进入当前会话的任务上下文

### `stats(): ApiResult`

- Endpoint: `GET /admin/merchant/v1/stats/session`

## 示例

```php
<?php

$created = $merchant->sessions()->create([
    'chat_mode' => 'model',
    'actor_type' => 'user',
    'actor_id' => '10001',
    'isnew' => true,
    'model_id' => 1,
]);

$session = $merchant->session('20260411S0A1B2C3D4E');
$detail = $session->get();
$signed = $session->sseSign(['expire_seconds' => 300]);
```
