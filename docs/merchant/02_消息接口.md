# Merchant 消息接口

## 根资源

类：`TaohuaHub\Ai2\Resource\Merchant\Messages`

### `getList(array $query = []): ApiResult`

- Endpoint: `GET /admin/merchant/v1/messages`
- 常用查询参数：
  - `page`
  - `pagesize`
  - `s_session_id`
  - `s_include_archive`
  - `s_request_id`
  - `s_message_role`
  - `s_content_type`
  - `s_keyword`

说明：

- 服务端要求消息列表必须提供 `s_session_id`
- 如果已经有 `session_id`，更推荐走 `session($id)->messages()->getList()`

### `get(string $messageId, bool $includeArchive = false): ApiResult`

- Endpoint: `GET /admin/merchant/v1/messages/detail`
- 查询参数：
  - `message_id`
  - `include_archive`

### `send(array $payload): ApiResult`

- Endpoint: `POST /admin/merchant/v1/messages/send`
- 主要参数：
  - `session_id`
  - `request_id`
  - `sender_actor_type`
  - `sender_actor_id`
  - `sender_actor_name`
  - `need_ai`
  - `content_type`
  - `content`
  - `attachments`

## 会话消息上下文

类：`TaohuaHub\Ai2\Context\Merchant\SessionMessages`

### `getList(array $query = []): ApiResult`

- Endpoint: `GET /admin/merchant/v1/messages`
- SDK 会强制注入当前会话的 `s_session_id`

### `get(string $messageId, bool $includeArchive = false): ApiResult`

- Endpoint: `GET /admin/merchant/v1/messages/detail`

### `send(array $payload): ApiResult`

- Endpoint: `POST /admin/merchant/v1/messages/send`
- SDK 会强制注入当前会话的 `session_id`

## 示例

```php
<?php

$result = $merchant->session('20260411S0A1B2C3D4E')->messages()->send([
    'request_id' => 'req_001',
    'sender_actor_type' => 'user',
    'sender_actor_id' => '10001',
    'content' => 'hello',
]);
```
