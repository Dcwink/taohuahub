# Merchant 任务接口

## 根资源

类：`TaohuaHub\Ai2\Resource\Merchant\Jobs`

### `getList(array $query = []): ApiResult`

- Endpoint: `GET /admin/merchant/v1/jobs`
- 常用查询参数：
  - `page`
  - `pagesize`
  - `s_session_id`
  - `s_job_id`
  - `s_request_id`
  - `s_queue_name`
  - `s_status`

### `get(string $jobId): ApiResult`

- Endpoint: `GET /admin/merchant/v1/jobs/detail`
- 查询参数：`job_id`

### `cancel(array $payload): ApiResult`

- Endpoint: `POST /admin/merchant/v1/jobs/cancel`
- 主要参数：
  - `job_id`
  - `reason`

## 会话任务上下文

类：`TaohuaHub\Ai2\Context\Merchant\SessionJobs`

### `getList(array $query = []): ApiResult`

- Endpoint: `GET /admin/merchant/v1/jobs`
- SDK 会强制注入当前会话的 `s_session_id`

### `get(string $jobId): ApiResult`

- Endpoint: `GET /admin/merchant/v1/jobs/detail`

### `cancel(array $payload): ApiResult`

- Endpoint: `POST /admin/merchant/v1/jobs/cancel`

## 示例

```php
<?php

$jobs = $merchant->session('20260411S0A1B2C3D4E')->jobs()->getList(['page' => 1]);
$detail = $merchant->jobs()->get('20260411J0A1B2C3D4E');
$cancel = $merchant->jobs()->cancel(['job_id' => '20260411J0A1B2C3D4E']);
```
