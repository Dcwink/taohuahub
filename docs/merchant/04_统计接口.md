# Merchant 统计接口

类：`TaohuaHub\Ai2\Resource\Merchant\Stats`

## `session(string $sessionId): ApiResult`

- Endpoint: `GET /admin/merchant/v1/stats/session`
- 查询参数：`session_id`

## `merchant(array $query = []): ApiResult`

- Endpoint: `GET /admin/merchant/v1/stats/merchant`
- 常用查询参数：
  - `stat_date`

## `session($id)->stats(): ApiResult`

- 这是 `SessionContext` 上的便利方法
- 内部仍调用 `GET /admin/merchant/v1/stats/session`

## 示例

```php
<?php

$sessionStats = $merchant->session('20260411S0A1B2C3D4E')->stats();
$merchantStats = $merchant->stats()->merchant(['stat_date' => '2026-04-13']);
```
