# System AI 会话接口

## 集合资源

类：`TaohuaHub\Ai2\Resource\System\Sessions`

### `create(array $payload): ApiResult`

- Endpoint: `POST /admin/system/v1/ai/sessions/create`
- 主要参数：
  - `chat_mode`
  - `actor_type`
  - `actor_id`
  - `actor_name`
  - `conversation_key`
  - `business_type`
  - `business_id`
  - `isnew`
  - `agent_id`
  - `model_id`
  - `mchno`

### `getList(array $query = []): ApiResult`

- Endpoint: `GET /admin/system/v1/ai/sessions`
- 常用查询参数：
  - `page`
  - `pagesize`
  - `s_session_id`
  - `s_mchno`
  - `s_actor_type`
  - `s_actor_id`
  - `s_conversation_key`
  - `s_chat_mode`
  - `s_status`

## 单会话上下文

类：`TaohuaHub\Ai2\Context\System\SessionContext`

### `id(): string`

- 返回当前绑定的 `session_id`

### `get(): ApiResult`

- Endpoint: `GET /admin/system/v1/ai/sessions/detail`

### `close(array $payload = []): ApiResult`

- Endpoint: `POST /admin/system/v1/ai/sessions/close`
- SDK 会强制注入当前上下文的 `session_id`

### `sseSign(array $payload = []): ApiResult`

- Endpoint: `POST /admin/system/v1/ai/sessions/sse-sign`
- SDK 会强制注入当前上下文的 `session_id`

### `switchModel(array $payload): ApiResult`

- Endpoint: `POST /admin/system/v1/ai/sessions/switch-model`
- 主要参数：`model_id`

### `messages(): SessionMessages`

- 进入当前 AI 会话的消息上下文

### `jobs(): SessionJobs`

- 进入当前 AI 会话的任务上下文
