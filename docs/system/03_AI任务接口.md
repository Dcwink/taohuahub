# System AI 任务接口

## 根资源

类：`TaohuaHub\Ai2\Resource\System\Jobs`

### `getList(array $query = []): ApiResult`

- Endpoint: `GET /admin/system/v1/ai/jobs`
- 常用查询参数：
  - `page`
  - `pagesize`
  - `s_session_id`
  - `s_job_id`
  - `s_request_id`
  - `s_queue_name`
  - `s_status`

### `get(string $jobId): ApiResult`

- Endpoint: `GET /admin/system/v1/ai/jobs/detail`

### `cancel(array $payload): ApiResult`

- Endpoint: `POST /admin/system/v1/ai/jobs/cancel`
- 主要参数：
  - `job_id`
  - `reason`
