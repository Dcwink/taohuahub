# System 商户接口

类：`TaohuaHub\Ai2\Resource\System\Merchants`

## 方法

### `create(array $payload): ApiResult`

- Endpoint: `POST /admin/system/v1/merchants/create`
- 主要参数：
  - `merchant_name`
  - `plan_code`
  - `discount_rate`
  - `status`
  - `expire_time`
  - `remark`

### `getList(array $query = []): ApiResult`

- Endpoint: `GET /admin/system/v1/merchants`
- 常用查询参数：
  - `page`
  - `pagesize`
  - `s_status`
  - `s_plan_code`
  - `s_keyword`

### `get(string $mchNo): ApiResult`

- Endpoint: `GET /admin/system/v1/merchants/detail`
- 查询参数：`mchno`

### `update(array $payload): ApiResult`

- Endpoint: `POST /admin/system/v1/merchants/update`

### `updateStatus(array $payload): ApiResult`

- Endpoint: `POST /admin/system/v1/merchants/status`

### `updateBalance(array $payload): ApiResult`

- Endpoint: `POST /admin/system/v1/merchants/balance`
- 主要参数：
  - `mchno`
  - `asset_type`
  - `change_type`
  - `amount`
  - `remark`
  - `operator`
  - `ref_request_id`

### `delete(array $payload): ApiResult`

- Endpoint: `POST /admin/system/v1/merchants/delete`

### `ledger(array $query = []): ApiResult`

- Endpoint: `GET /admin/system/v1/merchants/ledger`
- 常用查询参数：
  - `page`
  - `pagesize`
  - `s_mchno`
  - `s_year`
  - `s_sdate`
  - `s_edate`
  - `s_entry_type`
  - `s_asset_type`
  - `s_keyword`

### `ledgerDetail(string $ledgerId): ApiResult`

- Endpoint: `GET /admin/system/v1/merchants/ledger/detail`
- 查询参数：
  - `ledger_id`

说明：

- 服务端还支持 `year`
- SDK 当前用 `ledgerDetail($ledgerId)` 覆盖默认场景
- 如需指定 `year`，建议后续扩展方法签名
