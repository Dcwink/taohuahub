# System 运维接口

类：`TaohuaHub\Ai2\Resource\System\Ops`

## `queueStatus(): ApiResult`

- Endpoint: `GET /admin/system/v1/ops/queue-status`

## `workers(): ApiResult`

- Endpoint: `GET /admin/system/v1/ops/workers`

## `providerHealth(array $query = []): ApiResult`

- Endpoint: `GET /admin/system/v1/ops/provider-health`
- 常用查询参数：
  - `limit`

## `failedJobs(array $query = []): ApiResult`

- Endpoint: `GET /admin/system/v1/ops/failed-jobs`
- 常用查询参数：
  - `limit`
  - `s_source_system_code`
  - `s_mchid`

## `trace(array $query = []): ApiResult`

- Endpoint: `GET /admin/system/v1/ops/trace`
- 常用查询参数：
  - `s_request_id`
  - `s_session_id`
  - `s_job_id`

## 示例

```php
<?php

$ops = $system->ops();

$queue = $ops->queueStatus();
$workers = $ops->workers();
$failed = $ops->failedJobs(['limit' => 10]);
```
